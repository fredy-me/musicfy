# musicfy

Starter project layout for musicfy

Structure:

- assets/
  - css/style.css
  - images/singers/
  - audio/
  - video/
- config/db.php
- admin/add_singer.php
- admin/add_media.php
- pages/singer.php
- pages/album.php
- index.php

Place media files in `assets/audio` or `assets/video`. Update `config/db.php` with your DB credentials.
